# Portfolio Assets

Place your assets in this folder:

- `resume.pdf` - Your resume file (required)
- Project images/screenshots
- Logos or icons
- Any other media files

## Recommended Image Sizes

- **Project Screenshots**: 1200x800px or 16:9 aspect ratio
- **Profile Photo** (if adding): 400x400px square
- **OG Image** (for social sharing): 1200x630px

## Image Optimization

Before adding images:
1. Compress with [TinyPNG](https://tinypng.com/)
2. Consider using WebP format for better performance
3. Keep file sizes under 500KB when possible

## Resume

Add your resume as `resume.pdf` and it will be automatically linked from the download buttons on the site.
